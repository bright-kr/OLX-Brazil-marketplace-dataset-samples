이 **OLX Brazil Marketplace Ads Dataset Sample**에는 **1,000개 레코드**가 포함되어 있으며, **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `body` | 광고의 주요 내용 또는 설명입니다. |
| `subject` | 광고의 제목 또는 헤드라인입니다. |
| `Currency` | 가격이 표시되는 통화로, 일반적으로 BRL(브라질 헤알)입니다. |
| `priceValue` | 광고된 상품의 금액 또는 가격 값입니다. |
| `ProfessionalAnnouncement` | 광고가 전문 판매자에 의해 게시되었는지 여부를 나타냅니다(true/false). |
| `category` | 광고의 특정 카테고리를 나타내는 숫자 ID입니다. |
| `parentCategoryName` | 광고가 등록된 상위 카테고리로, 예: 'Autos e peças'입니다. |
| `categoryName` | 'Carros, vans e utilitários'와 같은 광고의 특정 카테고리 이름입니다. |
| `PublicationDate` | 광고가 게시된 날짜 및 시간입니다. |
| `adReply` | 광고가 받은 답글 또는 응답 수입니다. |
| `friendlyUrl` | 온라인에서 광고를 확인하기 위한 URL 링크입니다. |
| `Sellername` | 광고를 게시한 판매자 또는 엔티티의 이름입니다. |
| `Destaques` | 광고의 하이라이트 또는 주요 특징입니다. |
| `phone` | 판매자가 제공한 연락처 전화번호입니다. |
| `images` | 광고와 연관된 이미지의 URL 또는 링크입니다. |
| `NumberOfImages` | 광고에 포함된 이미지 개수입니다. |
| `neighbourhood` | 상품이 उपलब्ध하거나 등록된 지역의 동네 위치입니다. |
| `municipality` | 광고가 게시된 자치구 또는 도시입니다. |
| `zipcode` | 광고 위치와 연관된 우편번호입니다. |
| `zone` | 자치구 내의 구역 또는 지역 분류입니다. |
| `region` | 광고가 등록된 더 넓은 지역 또는 주(state)입니다. |
| `vehicleSpecificData` | 해당되는 경우, 차량과 관련된 추가 사양입니다. |
| `pubSpecificData` | 광고와 연관된 게시 관련 데이터 또는 메타데이터입니다. |
| `trackingSpecificData` | 광고 성과 및 참여 지표를 추적하는 데 사용되는 데이터입니다. |
| `searchboxes` | 적용된 검색 필터 또는 기준에 대한 정보입니다. |
| `breadcrumbUrls` | 사이트 내에서 광고를 찾기 위한 내비게이션 breadcrumb URL입니다. |
| `tags` | 광고와 연관된 키워드 또는 태그입니다. |
| `featured` | 광고가 추천(Featured) 또는 강조 표시되는지 여부를 나타냅니다(true/false). |
| `vehicleTags` | 차량 광고에 특화된 태그입니다. |
| `description` | 광고에 동반되는 상세 설명 또는 서술입니다. |
| `listTime` | 광고가 최초로 등록된 타임스탬프입니다. |
| `isFeatured` | 광고가 현재 플랫폼에서 추천(Featured) 상태인지 여부를 나타내는 Boolean 값입니다. |
| `Marca` | 해당되는 경우, 차량의 브랜드 또는 제조사입니다. |
| `TipoDeVeiculo` | 광고되는 차량의 유형 또는 카테고리입니다. |
| `Quilometragem` | 차량에 적용되는 주행거리 또는 주행 킬로미터입니다. |
| `PotenciaDoMotor` | 차량의 엔진 출력 사양입니다. |
| `Combustível` | 차량이 사용하는 연료 유형입니다. |
| `ManufactureYear` | 차량 또는 상품의 제조 연도입니다. |
| `properties` | 광고와 관련된 추가 속성 데이터입니다. |
| `locations` | 광고와 연관된 지리적 위치 데이터입니다. |
| `url` | 온라인에서 광고에 접근하기 위한 직접 URL 링크입니다. |
| `adId` | 각 광고에 할당된 고유 식별자입니다. |
| `listId` | 광고 엔트리와 연관된 리스팅 ID입니다. |
| `seller_id` | 판매자 또는 광고주의 고유 식별자입니다. |
| `destaque` | 광고의 주요 하이라이트 또는 특징입니다. |
| `tipo_anunciante` | 개인 또는 회사 등 광고주의 유형입니다. |


## Use Cases

### 1. Market Trend Analysis
광고 데이터를 분석하여 브라질 온라인 마켓플레이스의 새로운 시장 트렌드를 파악하고, 전략적 비즈니스 의사결정에 활용할 수 있습니다.

### 2. Competitive Intelligence
데이터셋을 활용하여 경쟁사의 가격 및 광고 전략을 모니터링하고, 시장 포지셔닝에 대한 인사이트를 확보할 수 있습니다.

### 3. Consumer Behavior Insights
소비자 참여 지표와 광고 응답을 검토하여 선호도를 이해하고, 마케팅 전략을 최적화할 수 있습니다.
